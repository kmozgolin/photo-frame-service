# Сервис рамки для фото

Простой статический сервис для загрузки фотографии, задания ширины рамки по вертикали и горизонтали, выбора цвета рамки и просмотра результата в превью.

## Как использовать

1. Откройте `index.html` в браузере.
2. Загрузите изображение.
3. Установите ширину рамки по вертикали и горизонтали.
4. Выберите цвет: черный, белый, пипетка или автоподбор.
5. Нажмите `ОК`.
6. Выберите фотографию из галереи через поле `Загрузить фото`.
7. Настройте ширину рамки, цвет и нажмите `ОК`.
8. После применения рамки нажмите `Скачать результат`.
9. Скачайте готовое изображение через кнопку `Скачать результат` при необходимости.
## Android APK

В папке `photo-frame-service/android` создан минимальный Android-проект с WebView, который загружает текущий веб-интерфейс из `app/src/main/assets/www`.

Сборка APK доступна на машине с установленным Android Studio / Android SDK и Gradle.

Пример команды в папке `photo-frame-service/android`:

```bash
gradle assembleRelease
```

или откройте проект в Android Studio и соберите `app`.
> Важно: если вы видите окно с названием приложения “Miro”, это значит, что вы используете чужой или неверный Client ID. Создайте свой OAuth 2.0 Client ID и используйте его.
